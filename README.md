# รวมโจทย์ Pre-Freshy-2026

## แนวทางในการเพื่มข้อใหม่

1. สร้าง branch ใหม่
2. สร้างโฟลเดอร์ใหม่เป็นชื่อโจทย์ใน Exams หรือ Labs
3. ภายในโฟลเดอร์ที่สร้างขึ้นมาใหม่ให้สร้างไฟล์ Description.md และก็โฟลเดอร์ Testcase ขึ้นมา

   3.1 ภายใน Description.md ให้เขียนรายละเอียดของโจทย์ในรูปแบบของ Markdown (สามารถดูตัวอย่างจากข้อ Hello ComCamp 36 ได้)

   3.2 ภายในโฟลเดอร์ Testcase ให้ใช้ไฟล์ txt โดยมี Format คือ :

   1in.txt สำหรับ input ข้อ 1 และ 1out.txt สำหรับ output ข้อ 1

   2in.txt สำหรับ input ข้อ 2 และ 2out.txt สำหรับ output ข้อ 2
   .
   .
   .

   3.3 Sample Input/Output สามารถใส่ใน testcases ได้ และโปรด ระบุไว้ใน README.md ภายในโฟลเดอร์ Testcase

   3.4 input ถ้ารับหลายค่าให้เว้นบรรทัด ห้ามใช้ 1 2 3 เพราะ Python จะอ่าน input แบบ line by line

   3.5 สามารถใส่เฉลยมาได้ในโฟลเดอร์

4. สร้างเสร็จแล้ว push จากนั้นเปิด PR เลย เดี๋ยว Merge ให้
