# Inter Goose's Lesson

มีน้องห่านอินเตอร์เพิ่งบินลัดฟ้ามาถึงประเทศไทยมาร่วมเชียร์ทีมฟุตบอลโลกแต่ยังพูดภาษาไทยไม่ค่อยชำนาญ จึงมีเป้าหมายอยากฝึกอ่านเลข 0-9999 เป็นภาษาไทยให้ได้ โดยการอ่านมีหลักการดังนี้

1. ถ้าเป็นเลข 0 โดด ๆ ให้ห่านออกเสียงว่า "ศูนย์"
2. ถ้าเลข 1 อยู่ในหลักหน่วยและมีหลักอื่นอยู่ข้างหน้า (เช่น 11, 21, 101) ห่านต้องเปลี่ยนเสียงเป็น "เอ็ด" เสมอ ห้ามพูดว่า หนึ่งเด็ดขาด
3. ถ้าเลข 2 อยู่ในหลักสิบ ให้ห่านออกเสียงว่า "ยี่สิบ" เท่านั้น

Input: จำนวนเต็ม N บรรทัดเดียวจากแป้นพิมพ์ โดยค่าปกติจะอยู่ในช่วง 0 <= N <= 9999

Output:

* แสดงผลลัพธ์เป็นคำอ่านภาษาไทยตามกฎการฝึกภาษาของน้องห่านอินเตอร์
* ทุกคำต้องเขียนติดกันโดยไม่มีการเว้นวรรคระหว่างคำ
* ในกรณีที่กรอกข้อมูลตัวเลขเกิน 9999 ให้แสดงคำว่า "Invalid"

Note: ระวังเรื่องคำยกเว้นของหลักสิบ (เช่น สิบ, ยี่สิบ) และให้ขึ้นบรรทัดใหม่ทุกรอบ (\n)
