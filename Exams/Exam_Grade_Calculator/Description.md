# Grade Calculator

**Difficulty:** Medium

## Problem

จงเขียนโปรแกรมภาษา Python เพื่อตัดเกรดนักเรียน โดยต้องสร้างฟังก์ชัน `calculate_grade(score)` เพื่อคำนวณเกรด จากนั้นรับจำนวนนักเรียนและคะแนนของแต่ละคน เพื่อแปลงคะแนนเป็นเกรด
เกณฑ์การให้คะแนน:
- 80 ขึ้นไป ได้เกรด A
- 70-79 ได้เกรด B
- 60-69 ได้เกรด C
- 50-59 ได้เกรด D
- ต่ำกว่า 50 ได้เกรด F

## Input

บรรทัดแรก: จำนวนเต็ม `N` ระบุจำนวนนักเรียน
บรรทัดที่ 2 ถึง `N+1`: บรรทัดละ 1 ข้อมูล ประกอบด้วย ชื่อนักเรียน ตามด้วยช่องว่าง และคะแนนที่เป็นจำนวนเต็ม (เช่น `Alice 85`)

## Output

บรรทัดละ 1 ข้อความ: พิมพ์ชื่อนักเรียน ตามด้วยช่องว่าง และเกรดที่ได้ ตัวอย่างเช่น:
Alice A
Bob F
