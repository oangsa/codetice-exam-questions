# Goose World Cup Average

ห่านน้อยเพิ่งกลับมาจากพักกลางวัน และรู้สึกอยากคำนวณแต้มบอลโลก แต่!!! ห่านน้อยเป็นชายชาตรีหล่อล้ำ สาว ๆ เห็นแล้วต้องตะลึง หนุ่ม ๆ ยังต้องหลงรัก ห่านน้อยจึงมีกฎการคำนวณคะแนนเฉลี่ยสะสมอยู่ ดังนี้

1. ห่านน้อยจะป้อนคะแนนการทำประตูของแต่ละแมตช์เข้าระบบไปเรื่อย ๆ อย่างต่อเนื่อง
2. หากห่านน้อยคีย์ตัวเลข `-1` เข้ามาเมื่อไหร่ ห่านน้อยจะคิดทันทีว่าคำนวณเสร็จสิ้นแล้ว และจะสั่งให้โปรแกรมหยุดทำงานเพื่อสรุปผลทันที

### Input
จำนวนจริง (Float) ป้อนเข้ามาเรื่อย ๆ จนกว่าจะเจอเลข -1 จึงจะหยุดรับค่า (ไม่นับเลข -1 มารวมในการคิดค่าเฉลี่ย)

### Output
แสดงค่าเฉลี่ยของชุดตัวเลขทั้งหมดที่รับเข้ามา โดยแสดงผลเป็นทศนิยม 2 ตำแหน่ง

### Note
ระวังเรื่องทศนิยม และให้ขึ้นบรรทัดใหม่ทุกรอบ (\n)
