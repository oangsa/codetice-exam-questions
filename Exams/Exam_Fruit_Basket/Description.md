# Fruit Basket

**Difficulty:** Easy

## Problem

จงเขียนโปรแกรมภาษา Python เพื่อรับรายการผลไม้ที่คั่นด้วยเครื่องหมายลูกน้ำ (`,`) และให้นับจำนวนของผลไม้แต่ละชนิด จากนั้นนำข้อมูลมาเก็บไว้ใน Dictionary แล้วแสดงผลลัพธ์ออกมาทีละรายการ

## Input

บรรทัดเดียว: ข้อความ String ที่ประกอบด้วยชื่อผลไม้คั่นด้วยลูกน้ำ (ตัวอักษรพิมพ์เล็กทั้งหมด) และจะไม่มีช่องว่างแทรก ตัวอย่างเช่น `apple,banana,apple,orange`

## Output

บรรทัดละ 1 ข้อความ: พิมพ์ชื่อผลไม้ ตามด้วยจำนวนที่นับได้ โดยแยกด้วยช่องว่าง (เรียงลำดับตามที่ปรากฏครั้งแรก) ตัวอย่างเช่น:
apple 2
banana 1
orange 1
