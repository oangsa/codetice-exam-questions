# ตรวจสอบคะแนนนักเตะบอลโลก 

ห่านน้อยรู้สึกอยากวิเคราะห์ฟอร์มการเล่นของนักเตะในศึกฟุตบอลโลก ห่านน้อยจึงมีกฎการเช็กคะแนนนักเตะอยู่ ดังนี้

1. รับชื่อนักฟุตบอลเข้ามาเรื่อย ๆ เพื่อบันทึกเป็น **List** จนกว่าจะเจอคำว่า `"STOP"` (ไม่นำคำว่า STOP มารวมในลิสต์)
2. รับชื่อนักฟุตบอลเพิ่มเข้ามาอีก 1 ชื่อ เพื่อเป็นชื่อที่ต้องการนำมาค้นหาและคำนวณคะแนน
3. ดึงคะแนนจาก **Dictionary** ของนักเตะคนนั้นมาคำนวณ โดยให้นับว่าในลิสต์มีชื่อนักเตะคนนี้อยู่ทั้งหมดกี่ครั้ง แล้วนำจำนวนครั้งไปคูณกับคะแนนเรตติ้งประจำตัว เพื่อหาคะแนนรวมทั้งหมด
4. ใช้เงื่อนไข **If-Else** เพื่อตรวจสอบว่าคะแนนรวมที่คำนวณได้นั้นเป็น **เลขคู่ (Even)** หรือ **เลขคี่ (Odd)** 

**Input:** 
* บรรทัดแรกเป็นต้นไป: ชื่อนักฟุตบอลทีละคน จนกว่าจะพิมพ์คำว่า `"STOP"`
* บรรทัดสุดท้าย: ชื่อนักฟุตบอล 1 คนที่ต้องการนำมาค้นหาคะแนนรวม

**Output:**
* แสดงข้อความ `"Even"` หรือ `"Odd"` ตามผลลัพธ์ของคะแนนรวมที่คำนวณได้

**Note:** หากไม่พบชื่อนักเตะที่ต้องการหาเลย (คะแนนรวมเป็น 0) ให้ถือว่าเป็นเลขคู่ และให้ขึ้นบรรทัดใหม่ทุกรอบ (\n)
