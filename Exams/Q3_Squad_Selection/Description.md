# World Cup: Squad Selection

คุณเป็นผู้จัดการทีมชาติที่ต้องจัดการข้อมูลรายชื่อนักเตะในทีม เนื่องจากมีนักเตะหลายคน คุณจึงต้องการจัดกลุ่มนักเตะตามตำแหน่งที่เล่น เพื่อให้ดูได้ง่ายขึ้น

**คำสั่ง:**
จงเขียนโปรแกรมรับข้อมูลรายชื่อนักเตะและตำแหน่งของนักเตะแต่ละคน จากนั้นจัดกลุ่มนักเตะตามตำแหน่ง และแสดงผลลัพธ์ออกมา

**ข้อมูลนำเข้า (Input):**
- บรรทัดแรก: จำนวนเต็ม $N$ หมายถึงจำนวนนักเตะในทีม
- $N$ บรรทัดถัดไป: ข้อมูลนักเตะ 1 คนต่อบรรทัด โดยมีรูปแบบคือ `ชื่อนักเตะ ตำแหน่ง` (คั่นด้วยช่องว่าง)

**ข้อมูลส่งออก (Output):**
- แสดงรายชื่อตำแหน่งที่นักเตะเล่น โดยเรียงลำดับการแสดงผลตามลำดับแรกของตำแหน่งที่รับเข้ามา
- ในแต่ละตำแหน่ง ให้แสดงผลรูปแบบ `Position: <ตำแหน่ง>`
- บรรทัดถัดมาให้แสดงรายชื่อนักเตะในตำแหน่งนั้นๆ โดยมีเครื่องหมาย `- ` นำหน้า (เรียงตามลำดับที่กรอกเข้ามา)

**ตัวอย่าง Input 1:**
```text
5
Messi Forward
Alisson Goalkeeper
Neymar Forward
Silva Defender
Ederson Goalkeeper
```

**ตัวอย่าง Output 1:**
```text
Position: Forward
- Messi
- Neymar
Position: Goalkeeper
- Alisson
- Ederson
Position: Defender
- Silva
```
