# World Cup: Transfer Budget

ในฐานะบอร์ดบริหารของสโมสรฟุตบอล คุณได้รับงบประมาณมาจำกัด คุณได้รับรายการข้อมูลนักเตะเป้าหมายที่สโมสรเล็งไว้และราคาค่าตัวในรูปแบบ JSON
ผู้จัดการทีมจะส่งรายชื่อนักเตะที่เขาอยากซื้อมาให้ คุณต้องเขียนโปรแกรมคำนวณว่า เรามีเงินพอซื้อนักเตะเหล่านั้นครบทุกคนหรือไม่

**คำสั่ง:**
ให้ใช้ module `json` ในการแปลงข้อมูล และนำราคามาคำนวณเปรียบเทียบกับงบประมาณ

**ข้อมูลนำเข้า (Input):**
- บรรทัดแรก: จำนวนเต็ม $B$ หมายถึง งบประมาณที่สโมสรมี
- บรรทัดที่ 2: ข้อความรูปแบบ JSON (Dictionary) ที่มี Key เป็นชื่อนักเตะเป้าหมาย และ Value เป็นราคาค่าตัว
- บรรทัดที่ 3: ชื่อนักเตะที่ผู้จัดการทีมต้องการซื้อ (คั่นด้วยช่องว่าง)

*หมายเหตุ: รับประกันว่าชื่อนักเตะที่ต้องการซื้อ จะอยู่ในรายการ JSON แน่นอน*

**ข้อมูลส่งออก (Output):**
- ถ้างบประมาณพอซื้อทุกคน ให้พิมพ์ `Deal! You bought everyone for <รวมราคาค่าตัว>. Remaining budget: <งบที่เหลือ>`
- ถ้างบประมาณไม่พอ ให้พิมพ์ `Deal Failed! Not enough budget.`

**ตัวอย่าง Input 1:**
```text
200
{"Ronaldo": 50, "Neymar": 80, "Mbappe": 120, "Haaland": 90}
Ronaldo Neymar
```

**ตัวอย่าง Output 1:**
```text
Deal! You bought everyone for 130. Remaining budget: 70
```

**ตัวอย่าง Input 2:**
```text
150
{"Ronaldo": 50, "Neymar": 80, "Mbappe": 120, "Haaland": 90}
Neymar Mbappe
```

**ตัวอย่าง Output 2:**
```text
Deal Failed! Not enough budget.
```
