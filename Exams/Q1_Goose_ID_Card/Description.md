# บัตรประจำตัวห่าน

## Problem

สถาบัน Goose Academy ต้องการสร้างบัตรประจำตัวอย่างง่ายให้กับนักเรียนห่านแต่ละตัว

ให้เขียนโปรแกรมภาษา Python เพื่อรับข้อมูลพื้นฐานของห่าน 1 ตัว แล้วแสดงผลเป็นบัตรประจำตัวห่านตามรูปแบบที่กำหนด

## Input

ข้อมูลนำเข้ามีทั้งหมด 4 บรรทัด:

1. ชื่อของห่าน เป็นข้อความ
2. อายุของห่าน เป็นจำนวนเต็ม
3. ความยาวปีก เป็นจำนวนทศนิยม
4. สถานะกัปตัน เป็นข้อความ `True` หรือ `False`

## Output

แสดงบัตรประจำตัวห่านตามรูปแบบที่กำหนดให้ตรงกับตัวอย่าง

## Notes

- รูปแบบผลลัพธ์ต้องตรงตามตัวอย่าง
- Float ใช้ ทศนิยม 2 ตัว
- สถานะกัปตันให้แสดงผลตรงตามค่าที่รับเข้ามา
