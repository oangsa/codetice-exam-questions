# Goose World Cup Factorial Sum

ห่านน้อยเพิ่งกลับมาจากพักกลางวัน และรู้สึกอยากคำนวณแต้มรวมพลังแฟกทอเรียลของลูกโทษบอลโลก แต่!!! ห่านน้อยเป็นชายชาตรีหล่อล้ำ สาว ๆ เห็นแล้วต้องตะลึง หนุ่ม ๆ ยังต้องหลงรัก ห่านน้อยจึงมีกฎการคำนวณระดับเทพจุติอยู่ ดังนี้

1. ห่านน้อยจะป้อนจำนวนลูกโทษสำเร็จในแต่ละแมตช์เข้าระบบไปเรื่อย ๆ อย่างต่อเนื่อง และระบบต้องนำเลขนั้นไปหาค่าแฟกทอเรียล ($n!$) เพื่อเก็บสะสมไว้
2. หากห่านน้อยคีย์ข้อความคำว่า `END` เข้ามาเมื่อไหร่ ห่านน้อยจะคิดทันทีว่าคำนวณเสร็จสิ้นแล้ว และจะสั่งให้โปรแกรมหยุดทำงานเพื่อรวมผลลัพธ์แฟกทอเรียลทั้งหมดทันที

** แฟกทอเรียล ($n!$) คืออะไร? (ห่านน้อยขอติวให้):**
มันคือผลคูณของจำนวนเต็มบวกตั้งแต่ 1 ไปจนถึง $n$ เช่น:
* $3! = 3 \times 2 \times 1 = 6$
* $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$
* **ความลับระดับมาดเท่:** ค่าของ $0!$ และ $1!$ จะมีค่าเท่ากับ `1` เสมอ!

**ข้อกำหนดพิเศษระดับชายชาตรี:** * ให้นักศึกษาเขียนฟังก์ชันสำหรับคำนวณหาค่า Factorial ขึ้นมาใช้งานเองด้วยลำแข้ง 
* ห้ามใช้การ `import` โมดูลสำเร็จรูปใด ๆ ทั้งสิ้น (เช่น ห้ามใช้ `import math`) ถ้าห่านน้อยเห็นคำว่า `import` แม้แต่ตัวเดียว จะถือว่าไม่แมนและไม่ได้คะแนนในข้อนี้!

### Input
จำนวนเต็มบวก $n$ ($0 \le n \le 12$) ป้อนเข้ามาเรื่อย ๆ บรรทัดละ 1 ตัว จนกว่าจะเจอข้อความคำว่า `END` จึงจะหยุดรับค่า

### Output
แสดงผลรวมของค่าแฟกทอเรียลทั้งหมดที่คำนวณได้ จากชุดตัวเลขที่รับเข้ามา

### Note
ระวังกรณีที่อินพุตเป็นเลข 0 โดย $0! = 1$ และให้ขึ้นบรรทัดใหม่ทุกรอบ (\n)
