# Goose World Cup Factorial Sum ห่านน้อยรวมพลังแฟกทอเรียล

###  รายละเอียดโจทย์
ห่านน้อยกำลังคลั่งไคล้สถิติการเตะลูกโทษในฟุตบอลโลก และอยากสร้างดัชนีชี้วัดความเทพของแต่ละทีม โดยมีกฎคือ ทุกครั้งที่ป้อนจำนวนลูกโทษที่ทีมทำได้เข้ามา ให้โปรแกรมนำจำนวนนั้นไปหาค่าแฟกทอเรียล ($n!$) เพื่อเก็บสะสมไว้ ห่านน้อยจะป้อนจำนวนตัวเลขเข้ามาเรื่อย ๆ จนกว่าจะพอใจแล้วพิมพ์ข้อความคำว่า `END` ระบบจึงจะหยุดรับค่า จากนั้นให้โปรแกรมทำการรวมผลลัพธ์แฟกทอเรียลของทุกตัวที่บันทึกไว้ แล้วแสดงผลลัพธ์รวมออกมา

**ข้อกำหนดพิเศษ:** 
* ให้เขียนฟังก์ชันสำหรับคำนวณหาค่า Factorial ขึ้นมาใช้งานเอง 
* ห้ามใช้การ `import` โมดูลสำเร็จรูปใด ๆ ทั้งสิ้น (เช่น ห้ามใช้ `import math`) หากตรวจพบจะไม่ได้คะแนนในข้อนี้

### Input
จำนวนเต็มบวก $n$ ($0 \le n \le 12$) ป้อนเข้ามาเรื่อย ๆ บรรทัดละ 1 ตัว จนกว่าจะเจอข้อความคำว่า `END` จึงจะหยุดรับค่า

### Output
แสดงผลรวมของค่าแฟกทอเรียลทั้งหมดที่คำนวณได้ จากชุดตัวเลขที่รับเข้ามา

### Note
ให้ขึ้นบรรทัดใหม่ทุกรอบ (\n) และระวังกรณีที่อินพุตเป็นเลข 0 โดย $0! = 1$
