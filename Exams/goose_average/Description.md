# Goose World Cup Average ห่านน้อยคำนวณแต้มบอลโลก

### 📝 รายละเอียดโจทย์
ห่านน้อยกำลังนั่งชมการแข่งขันฟุตบอลโลก และต้องการคำนวณหาค่าเฉลี่ยของคะแนน (หรือแต้มทำประตู) ของทีมที่ชอบในแต่ละนัด โดยห่านน้อยจะนั่งพิมพ์คะแนนที่ทีมทำได้ในแต่ละแมตช์ลงระบบไปเรื่อย ๆ อย่างต่อเนื่อง แต่ถ้าเมื่อไหร่ที่ห่านน้อยพิมพ์ตัวเลข `-1` แปลว่าห่านน้อยดูบอลจบแล้วและต้องการให้ระบบหยุดรับค่าทันที

ให้นักศึกษาเขียนโปรแกรมเพื่อรับคะแนนเข้ามา และคำนวณหาค่าเฉลี่ยของคะแนนฟุตบอลทั้งหมดที่ห่านน้อยป้อนเข้ามา (โดยไม่นำเลข `-1` ที่ใช้ปิดโปรแกรมมารวมในการคิดค่าเฉลี่ย)

*หมายเหตุ: ทศนิยมของผลลัพธ์ให้แสดงผลเป็น 2 ตำแหน่ง และให้ขึ้นบรรทัดใหม่ (\n) เสมอ*

### 📥 Input
* บรรทัดที่ 1 ถึง บรรทัดที่ $N$: จำนวนจริง (Float) แทนคะแนนที่ป้อนเข้ามาบรรทัดละ 1 ตัว
* บรรทัดสุดท้าย: เลข `-1` (เมื่อระบบเจอค่านี้จะหยุดอ่านข้อมูลทันที)

### 📤 Output
* จำนวนจริง 1 ค่า แสดงค่าเฉลี่ยของคะแนนทั้งหมดที่ป้อนเข้ามา (แสดงทศนิยม 2 ตำแหน่ง)
