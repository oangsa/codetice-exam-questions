# World Cup: สรุปเวลาฝึกซ้อม

## Problem

ทีมฟุตบอลกำลังเตรียมตัวสำหรับการแข่งขัน World Cup

โค้ชต้องการคำนวณเวลาฝึกซ้อมรวมของทีมก่อนเริ่มการแข่งขัน

ให้เขียนโปรแกรมภาษา Python เพื่อรับข้อมูลดังนี้:

1. จำนวนวันฝึกซ้อม
2. จำนวนชั่วโมงฝึกซ้อมต่อวัน
3. จำนวนเวลาพักต่อวัน หน่วยเป็นนาที

จากนั้นคำนวณและแสดงผล:

- จำนวนชั่วโมงฝึกซ้อมทั้งหมดก่อนหักเวลาพัก
- จำนวนชั่วโมงพักทั้งหมด
- จำนวนชั่วโมงฝึกซ้อมจริงหลังหักเวลาพัก

ใช้สูตรดังนี้:

```txt
total_training_hours = training_days * training_hours_per_day
total_break_hours = training_days * break_minutes_per_day / 60
actual_training_hours = total_training_hours - total_break_hours
```

## Input

ข้อมูลนำเข้ามีทั้งหมด 3 บรรทัด:

1. จำนวนวันฝึกซ้อม เป็นจำนวนเต็ม
2. จำนวนชั่วโมงฝึกซ้อมต่อวัน เป็นจำนวนทศนิยม
3. จำนวนเวลาพักต่อวัน หน่วยเป็นนาที เป็นจำนวนเต็ม

## Output

แสดงสรุปเวลาฝึกซ้อมตามรูปแบบที่กำหนดให้ตรงกับตัวอย่าง

## Notes

- รูปแบบผลลัพธ์ต้องตรงตามตัวอย่าง
- ค่าที่คำนวณได้สามารถแสดงเป็นเลขทศนิยม 2 ตำแหน่ง
