# Library Book

**Difficulty:** Hard

## Problem

จงเขียนโปรแกรมภาษา Python จำลองการยืมคืนหนังสือในห้องสมุด โดยให้สร้าง Class ชื่อ `Book` ที่มีโครงสร้างดังนี้:
1. ตัวแปรในคลาส (Attributes): `title` (ชื่อหนังสือ) และ `is_available` (สถานะการยืม เริ่มต้นต้องเป็น `True`)
2. เมธอด (Methods):
   - `__init__(self, title)`: สำหรับรับชื่อหนังสือ
   - `borrow_book(self)`: ถ้า `is_available` เป็น `True` ให้เปลี่ยนเป็น `False` และคืนค่า `True` (ยืมสำเร็จ) ถ้าไม่ได้ให้คืนค่า `False` (ยืมไม่สำเร็จ)
   - `return_book(self)`: เปลี่ยน `is_available` กลับเป็น `True`

เมื่อโปรแกรมทำงาน จะรับข้อมูลชื่อหนังสือ และลำดับคำสั่ง (ยืม หรือ คืน) และแสดงผลลัพธ์การทำงานของแต่ละคำสั่ง

## Input

บรรทัดแรก: ชื่อหนังสือ (String)
บรรทัดที่สอง: จำนวนเต็ม `N` ระบุจำนวนคำสั่ง
บรรทัดที่ 3 ถึง `N+2`: คำสั่งแต่ละคำสั่ง ซึ่งจะเป็นคำว่า `borrow` หรือ `return`

## Output

แสดงผลการทำงานทีละคำสั่งตามบรรทัด:
- ถ้าเจอคำสั่ง `borrow`: หากยืมได้พิมพ์ว่า `Borrowed [ชื่อหนังสือ]` หากยืมไม่ได้พิมพ์ว่า `[ชื่อหนังสือ] is not available`
- ถ้าเจอคำสั่ง `return`: พิมพ์ว่า `Returned [ชื่อหนังสือ]`
