# Boy's Rule

ห่านน้อยเพิ่งกลับมาจากพักกลางวัน และรู้สึกอยากเข้าห้องน้ำ แต่!!! ห่านน้อยเป็นชายชาตรีหล่อล่ำ สาว ๆ เห็นแล้วต้องตะลึง หนุ่ม ๆ ยังต้องหลงรัก ห่านน้อยจึงมีกฎการเข้าห้องน้ำอยู่ ดังนี้

1. ถ้าโถข้าง ๆ ไม่ว่าง ห่านน้อยจะไม่เข้า
2. ถ้าห่านน้อยพบเห็นชายชาตรีใช้โถปัสสาวะ ข้าง ๆ กัน ห่านน้อยจะคิดทันทีว่า "Rule has been break" และจะไปใช้ห้องน้ำอื่นทันที

Input:
จำนวนเต็ม 0 และ 1 จนกว่าจะเจอเลข -1 จึงจะหยุดรับค่า (ไม่เกิน 999 ตัวเลข)

Output: 
- ถ้าไม่มีโถว่างเลยจะแสดงคำว่า "No urinals available" 
- ถ้าพบการแหกกฎจะแสดงคำว่า "Rule has been break" 
- นอกจากนั้นให้แสดงคำว่า "There is/are ... urinal(s) available"

Note: ระวังเรื่องแกรมม่า และให้ขึ้นบรรทัดใหม่ทุกรอบ (\n)
